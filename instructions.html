Wanna know to play?

First you're given a shuffled grid

Some tiles are fixed, use these fixed tiles as reference for how the grid should look.

Your main goal is to sort tiles so the colors 'flow'

You can drag and drop tiles. Switch tiles around by picking up one tile and dropping it into the tile you want to swap with. Try it on the board on the left.
